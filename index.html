live · PyPI
Universal Python Runtime Hypervisor. Daemon worker pool with filesystem isolation (bubbles), multi-version Python concurrency, zero-copy shared memory IPC, dynamic CUDA loading, and a C dispatcher binary for <1ms startup. Moving deeper into Rust for the FFI layer and C for the hot paths.

Rust FFI layer over uv. ~2.78× faster than the uv binary. 671+ wheels including exotic platforms. abi3 ready.

PEP 503 wheel index for platforms PyPI forgot. cryptography, psutil, cffi for musllinux armv7l, aarch64, ppc64le. Drop-in for pip — no custom tooling required.
